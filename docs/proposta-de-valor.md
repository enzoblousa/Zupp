# Proposta de Valor

O diferencial do produto está em duas frentes complementares.

## Para o cliente

Facilidade de encontrar advogados dispostos a resolver seu problema, com **proximidade geográfica** e **disponibilidade imediata**.

## Para o advogado

Facilidade de captar clientes geograficamente próximos que demandam serviços **no momento**.

## Posicionamento da plataforma

A plataforma atua como camada de intermediação que **reduz fricção entre as duas pontas** — eliminando a necessidade de rede de contatos prévia tanto para quem precisa do serviço quanto para quem o presta.
